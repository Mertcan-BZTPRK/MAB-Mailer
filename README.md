## 🇹🇷 Türkçe

MAB Mailer, kişileri organize edebileceğiniz, mesajları özelleştirebileceğiniz ve tek tıkla yüzlerce e-postayı sorunsuz bir şekilde gönderebileceğiniz **toplu e-posta gönderim masaüstü uygulamasıdır.**

### ✨ Özellikler

* <strong style="color: #0ea5e9;">📋 Excel'den Toplu Aktarım:</strong> Şablon oluşturun, doldurun ve kişileri anında içe aktarın.
* <strong style="color: #0ea5e9;">👥 Grup Yönetimi:</strong> Kişileri gruplayın, filtreleyin ve yalnızca hedef kitlenize e-posta gönderin.
* <strong style="color: #0ea5e9;">✉️ Kişiselleştirilmiş E-posta:</strong> `{Ad}`, `{Soyad}`, `{Email}` ve dinamik sütun değişkenleri kullanarak her alıcıya özel hitap edin.
* <strong style="color: #0ea5e9;">📝 Mesaj Şablonları:</strong> Sık gönderilen mesajları kaydedin ve tekrar tekrar kullanın.
* <strong style="color: #0ea5e9;">🔗 HTML Link:</strong> Mail içeriğine tıklanabilir bağlantıları kolayca ekleyin.
* <strong style="color: #0ea5e9;">✒️ İmza Desteği:</strong> Kurumsal veya kişisel e-posta imzanızı otomatik olarak ekleyin.
* <strong style="color: #0ea5e9;">📮 Kuyruk Sistemi:</strong> Gönderimleri kuyruğa alın, toplu başlatın ve anlık teslimat durumunu takip edin.
* <strong style="color: #0ea5e9;">🔄 Çoklu Hesap:</strong> Birden fazla SMTP hesabı kaydedip aralarında kolayca geçiş yapın.
* <strong style="color: #0ea5e9;">🌗 Otomatik Tema:</strong> Windows sisteminizin Açık / Koyu temasına dinamik olarak uyum sağlar.
* <strong style="color: #0ea5e9;">🔀 Spoofing:</strong> Gönderen adresini (SMTP sunucusu izin veriyorsa) dilediğiniz gibi özelleştirin.

## 🇬🇧 English

MAB Mailer is a **bulk email sending desktop application.** Organize contacts, personalize messages, and send hundreds of emails with one click flawlessly.

### ✨ Highlights & Features

* <strong style="color: #0ea5e9;">📋 Bulk Import from Excel:</strong> Generate a template, fill it in, and import contacts instantly.
* <strong style="color: #0ea5e9;">👥 Group Management:</strong> Organize contacts into groups, filter them, and send targeted emails efficiently.
* <strong style="color: #0ea5e9;">✉️ Personalized Emails:</strong> Use dynamic variables like `{Name}`, `{Surname}`, and `{Email}` to address hundreds of recipients personally in bulk.
* <strong style="color: #0ea5e9;">📝 Message Templates:</strong> Save and reuse frequently sent messages with a local draft database.
* <strong style="color: #0ea5e9;">🔗 HTML Links:</strong> Insert clickable hyperlinks directly into the email body from a special dialog window.
* <strong style="color: #0ea5e9;">✒️ Signature Support:</strong> Automatically append personalized HTML signatures to your emails.
* <strong style="color: #0ea5e9;">📮 Queue System:</strong> Queue drafts, bulk send them, and seamlessly track delivery status.
* <strong style="color: #0ea5e9;">🔄 Multi-Account:</strong> Store multiple SMTP accounts and switch between them easily from the settings panel.
* <strong style="color: #0ea5e9;">🌗 Auto Theme:</strong> Automatically follows your Windows Light or Dark theme settings natively.
* <strong style="color: #0ea5e9;">🔀 Spoofing:</strong> Customize the sender address (if your SMTP server allows it).

### 📦 Tech Stack / Teknolojiler

### 📁 Project Structure / Proje Yapısı

```text
MAB Mailer/
│
├── App.xaml/.cs                    # Entry point, theme loading
├── LoginWindow.xaml/.cs            # Login screen, account management
├── MainWindow.xaml/.cs             # Main screen, mail composition
│
├── DatabaseService.cs              # SQLite CRUD operations
├── MailService.cs                  # SMTP email sending
├── ExcelService.cs                 # Excel template & import
├── TemplateEngine.cs               # Variable parsing ({Name}, {Surname}...)
├── ThemeService.cs                 # Windows theme detection
├── GlobalSettings.cs               # Runtime session settings
│
├── CustomAlert.xaml/.cs            # Custom alert/confirm dialog
├── LinkWindow.xaml/.cs             # HTML link insertion dialog
├── TemplateNameWindow.xaml/.cs     # Template save dialog
├── InputBox.cs                     # Numeric input dialog
│
├── EmailAccount.cs                 # Account model
├── Recipient.cs                    # Recipient model
├── RecipientGroup.cs               # Group model
├── MailDraft.cs                    # Queue item model
└── MessageTemplate.cs              # Message template model
```
